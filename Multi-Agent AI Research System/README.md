# Research Studio — Multi-Agent Research Assistant

A multi-agent research application that turns a user topic into a structured, AI-generated research report.

The project uses separate agents for web search and website scraping, then uses Gemini to write and review the final report. A polished Streamlit interface lets users follow each stage and inspect the exact search and scraped results used by the agents.

## Features

- 🔎 **Web research** using Tavily Search
- 📄 **Website scraping** with BeautifulSoup4
- ✍️ **AI report writing** using Google Gemini through LangChain
- ✅ **Report scoring and feedback** generated by a critic chain
- 📊 **Live workflow status** in the Streamlit interface
- 🧾 **Inspectable agent outputs** for both search and scraping stages
- 🎨 **Professional Streamlit interface** with a clean report workspace

## How It Works

The application follows a four-stage workflow:

1. **Searching Agent** — Searches for recent and reliable information on the topic using Tavily.
2. **Scraping Agent** — Selects a relevant result and extracts clean website text using BeautifulSoup.
3. **Writing Agent** — Uses the gathered search and scrape data to produce a structured report.
4. **Scoring Agent** — Reviews the report and returns a score, strengths, improvement areas, and verdict.

The Streamlit interface runs these stages directly, displays the current stage while it is running, and lets the user view the original results returned by `search_tool` and `scrape_tool`.

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository-name.git
cd your-repository-name
```

### 2. Create and activate a virtual environment

**Windows**

```bash
python -m venv .venv
.venv\Scripts\activate
```

**macOS / Linux**

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Add environment variables

Create a `.env` file in the project root:

```env
TAVILY_API_KEY=your_tavily_api_key_here
GOOGLE_API_KEY=your_google_gemini_api_key_here
```

> Never commit your `.env` file or API keys to GitHub.

## Run the Streamlit Interface

Start the application with:

```bash
streamlit run M4_interface.py
```

Streamlit will provide a local URL, normally:

```text
http://localhost:8501
```

Open it in your browser, enter a research topic, and click **Generate report**.

## Using the Interface

1. Type a research topic into the input field.
2. Click **Generate report**.
3. Watch the status panel as the search, scraping, writing, and scoring stages run.
4. Read the generated research report and report feedback.
5. Use the left-side buttons to inspect:
   - **View searching agent results**
   - **View scraping agent results**

## Running the Pipeline from Python

You can also run the original pipeline directly from a Python shell:

```python
from M3_pipeline import research_pipeline

research_pipeline("Impact of artificial intelligence on healthcare")
```
